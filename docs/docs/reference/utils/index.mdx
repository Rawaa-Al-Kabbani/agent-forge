This section provides reference documentation for utility classes and functions provided by Agent Forge that can be helpful for various tasks when building or extending agents.

Key utilities include:

*   **[`EventEmitter`](./event-emitter.mdx):** A simple class for implementing event-driven patterns.
*   **[`RateLimiter`](./rate-limiter.mdx):** A utility to control the frequency of operations.
*   **[`Streaming Utilities`](./streaming.mdx):** Helpers related to handling and visualizing streaming data, primarily the `enableConsoleStreaming()` function.

Refer to the specific pages for detailed API information.
